How this page works

Nobody types these numbers. A script on a server reads the live databases every hour — the outreach logs, the subscriber count, the AI spend meter, the essays shipped — renders this exact page, and commits it to a public Git repository. No hand-editing, no rounding up, no quietly skipping a bad day.

Don't take my word for it: every hourly commit is timestamped in the open — the machine's own paper trail is the proof the numbers aren't curated.
